# 🛡 NetSentinel - Cyber Port Scanner

A web-based network port scanner built using Python and Flask.

## Features

- TCP Port Scanner
- UDP Port Scanner
- Service Detection
- Banner Grabbing
- Scan Dashboard
- TXT Report Export
- CSV Report Export
- PDF Report Export
- Scan History Viewer

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript

## How to Run

Install dependencies:

pip install -r requirements.txt

Run:

python app.py

Open:

http://127.0.0.1:5000
